# DecorFlow

A full-stack MERN-based platform for managing event decoration services with booking, payments, and real-time tracking.

## Features
- Appointment booking, rescheduling, and cancellation  
- Secure payment with invoice generation  
- Decorator filtering by specialty  
- Real-time status tracking workflow  

## Tech Stack
- React, Tailwind CSS  
- Node.js, Express.js  
- MongoDB  
- JWT Authentication  
- Stripe API  

## Setup

Backend:
cd style-decor-server  
npm install  
npm start  

Frontend:
cd style-decor-client  
npm install  
npm run dev
